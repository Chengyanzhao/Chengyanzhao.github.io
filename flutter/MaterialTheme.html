# Scaffold

https://api.flutter.dev/flutter/material/Scaffold-class.html

参数 参数类型 说明 可选or必选
key Key - -
primary bool 是否在屏幕顶端显示此Scaffold，默认为true 可选
appBar PreferredSizeWidget 顶部的标题栏，不设置的话就不会显示 可选
backgroundColor Color 内容区域背景颜色 可选
body Widget 显示内容 可选
bottomNavigationBar Widget 显示在页面底部的导航栏，通常使用BottomNavigationBar 可选
bottomSheet Widget 显示在底部的内容，不跟随屏幕滚动，固定在下方展示。 可选
floatingActionButton Widget 固定在右下角的悬浮按钮，通常使用FloatingActionButton。 可选
floatingActionButtonLocation FloatingActionButtonLocation 确定floatingActionButton位置。默认FloatingActionButtonLocation.endFloat。 可选
floatingActionButtonAnimator FloatingActionButtonAnimator 悬浮按钮floatingActionButton在[floatingActionButtonLocation]出现/消失动画。默认FloatingActionButtonAnimator.scaling 可选
persistentFooterButtons List<Widget> 固定在下方显示的按钮列表，比如对话框下方的确定、取消按钮。 可选
drawer Widget 侧边栏控件，从左向右滑出。通常使用Drawer。 可选
endDrawer Widget 侧边栏控件，与draser相同，只是方向相反，从右向左滑出。 可选
drawerDragStartBehavior DrawerDragStartBehavior 确定处理拖动开始行为的方式。 可选
drawerScrimColor Color 抽屉打开时用来遮盖主要内容的涂布颜色，默认 Colors.black54. 可选
extendBody bool 确定body范围。值为true时，body延伸到Scaffold底部，包括bottomNavigationBar 和 persistentFooterButtons区域。值为false时相反，body只延伸到bottomNavigationBar 和 persistentFooterButtons的顶部。 可选
resizeToAvoidButtonPadding bool 控制界面内容body是否重新布局来避免底部被覆盖，比如当键盘显示的时候，重新布局避免被键盘盖住内容。默认为true 可选

# BottomNavigationBar

底部导航条，一般用于Scaffold内的bottomNavigationBar属性。

参数 参数类型 说明 可选or必选
key Key Widget标识 可选
items List<BottomNavigationBarItem> 底部导航栏的选项列表 必选
onTap ValueChanged<int> 底部导航栏选项的点击事件，返回的值是选项的 index 可选
currentIndex int 当前选中的选项的 index 可选
type BottomNavigationBarType 定义底部导航栏的选项如何显示，有两个值：BottomNavigationBarType.fix：当选项少于4个时，默认为这个类型，如果 fixedColor不为空，则选项选中时的颜色就是 fixedColor，否则是 ThemeData.primaryColor，导航栏背景颜色是 ThemeData.canvasColor。BottomNavigationBarType.shifting：当选项不小于4个时，默认为这个类型，所有的选项都会被渲染为白色，但是导航栏的背景色是当前选中的选项的背景色 可选
fixedColor Key 当 BottomNavigationBarType 的 type 是 fix 时，选中的选项的颜色就是 fixdColor 可选
iconSize double BottomNavigationBarItem 的 icon 的大小 可选
