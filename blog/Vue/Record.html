# 浏览器不支持Promise

Vue项目打包上线测试时，部分老旧浏览器(主要是IE9)会报错：

WARNING

Unhandled promise rejection ReferenceError: “Promise”未定义

  • 原因

项目使用Promise，而浏览器未支持。

  • 解决办法

安装babel-polyfill模块，在项目main.js中引用，或者在webpack中配置。

npm i babel-polyfill -S

import 'babel-polyfill'
import Vue from 'Vue'
// other codes

// webpack config
module.exports = {
  entry: {
    app: ['babel-polyfill', './app/js']
  }
}
